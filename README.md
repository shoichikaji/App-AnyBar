# NAME

App::AnyBar - a perl client for AnyBar

# SYNOPSIS

    use App::AnyBar;
    my $anybar = App::AnyBar->new;
    $anybar->color('red');
    $anybar->quit;

# DESCRIPTION

App::AnyBar is a perl client for [https://github.com/tonsky/AnyBar](https://github.com/tonsky/AnyBar).
